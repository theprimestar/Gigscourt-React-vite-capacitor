# GigsCourt

Local service marketplace app.

## Tech Stack
- Vite + React
- Capacitor (Android + iOS)
- Supabase (Auth)
- Firebase (Real-time)

## Development
- Web: [Vercel Deployment](https://gigscourt.vercel.app)
- `npm run dev` — local dev server
- `npm run build` — production build

## Native Builds
- Android: Manually trigger `build-android.yml` workflow
- iOS: Manually trigger `build-ios.yml` workflow, sign IPA with ESign
